INTERFACE PERSONALIZACIÓN PARLANTES

Este es mi proyecto final del ramo Programación para Diseño, la idea principal fue generar una herramienta que se pudiera implementar en un servicio de parlantes personalizados, teniendo parametros como cantidad, forma y detalles especificos.

A pesar de no estar muy bien implementado y de no llegar a tener el nivel de complejidad que uno esperaría, termine bastante conforme con el resultado, ademas de se una instancia para poner a prueba todo lo aprendido durante el curso.
